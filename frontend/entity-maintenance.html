Entity Maintenance

Retire stores or components to hide them from monitoring views. Retired entities reappear automatically when a new event is received. Event history is always preserved.

  • Store ID only — hides all components for that store.
  • Store ID + Component — hides that specific store/component pair.
  • Component only — hides that component across every store that currently reports it.
